OKF Harness 不是又一个笔记应用。它是藏在 Agent 背后的知识库引擎——你对 Agent 说自然语言，它帮你收集资料、整理成结构化 Wiki、随问随答。没有云端，没有账号，所有数据安安静静留在你自己的 Mac 上。

你能让它做什么

  • 初始化知识库：一句话创建完整工作空间，目录、索引、Agent 配置全部就位
  • 添加资料：拖进 PDF、Markdown、网页链接，自动归档、校验、登记
  • 整理吸收：Agent 读懂你的资料，生成摘要页、更新主题页、补全引用
  • 搜索提问：用自然语言问你的知识库，Agent 查索引、读全文、给出带来源的回答
  • 健康检查：一键扫描坏链接、缺引用、过期页面，知识库始终可用
  • 可视化图谱：生成本地 HTML 知识图谱，看清概念之间的关联

它跟其他工具有什么不同

  • 没有 GUI，Agent 是界面：你不是在点点点，是在跟 Agent 对话
  • 数据是你的，不是云的：所有 markdown 文件都在本地文件夹里，用 Git 管理，任何编辑器都能打开
  • OKF 格式，不锁定：Wiki 遵循 Open Knowledge Format，随时可以带走，不绑定任何特定工具
  • 三层结构，清清楚楚：原始资料（不可改）→ Wiki 正文（Agent 维护）→ 工作流指令（可审计）
  • CLI 是后门，不是正门：普通用户永远不需要记命令；高级用户可以用命令行做自动化
  • 开源透明：Apache-2.0 协议，代码可查、可改、可审计
